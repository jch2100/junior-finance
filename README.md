# 재무설계 기초 — 주니어를 위한 첫걸음 (HTML 슬라이드 덱)

신입·주니어 사원 대상 **재무설계 교육**용 HTML 프레젠테이션입니다.
Simon Sinek의 **Golden Circle(Why → How → What)** 구조를 차용해
"왜 돈을 모으는가"에서 출발합니다.

## 여는 법
`index.html` 파일을 **더블클릭**하면 브라우저에서 바로 열립니다.

- 외부 라이브러리·CDN·인터넷 연결이 **전혀 필요 없습니다.** (단일 파일, 오프라인 완벽 동작)
- CSS·JS·아이콘(SVG)이 모두 `index.html` 한 파일에 내장되어 있습니다.

## 조작 키
| 동작 | 키 / 제스처 |
|---|---|
| 다음 슬라이드 | `→` · `Space` · `PageDown` · 화면 우측 클릭 · 왼쪽으로 스와이프 |
| 이전 슬라이드 | `←` · `PageUp` · 화면 좌측 클릭 · 오른쪽으로 스와이프 |
| 처음 / 끝 | `Home` / `End` |
| 전체화면 토글 | `F` |

하단에 **섹션 라벨 · 진행 막대 · 슬라이드 번호**가 표시됩니다.
URL 끝의 `#번호`(예: `index.html#8`)로 특정 슬라이드에 바로 진입할 수 있습니다.

## 슬라이드 추가 / 편집하는 법
각 슬라이드는 아래와 같은 한 개의 `<section>` 블록입니다.

```html
<section class="slide" data-section="2. HOW — 새 주제">
  <span class="eyebrow">라벨</span>
  <h2>슬라이드 제목</h2>
  <ul class="clean slide-body">
    <li><strong>핵심</strong> — 설명 문장</li>
  </ul>
</section>
```

- `index.html` 안에서 `슬라이드 15` 주석 위치에 블록을 복사해 넣으면 됩니다.
- `data-section` 값이 하단 섹션 라벨로 표시됩니다.
- 슬라이드 번호·카운터·진행 막대는 **자동 계산**되므로 따로 손댈 필요가 없습니다.

### 자주 쓰는 구성 요소 (CSS 클래스)
- `divider` — 섹션 표지(레드 배경) / `cover` — 표지 / `reflect` — 성찰형 질문
- `grid c2` · `grid c3` + `card` (`accent-card`) — 카드 레이아웃
- `ul.clean` — 불릿 목록 / `formula` · `branch` — 공식·갈래 도식
- `placeholder` — 작성 예정 자리

## 디자인 토큰
`:root`의 CSS 변수로 색·폰트를 한 번에 바꿀 수 있습니다.
미니멀 · 아이보리 배경(`--bg`) · 톤다운 레드 포인트(`--accent: #B23A35`).
